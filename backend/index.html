    Chatur Backend¶

    The Chatur Backend includes virtual machines running on a self-managed OpenStack Cloud.

    We'll use the analogy of building a digital boat for the project.

    Chatur is built of a combination of open-source projects including:

    Backend - Rudder LLM - Sails/Propellers UI - Crow's Nest UX - Anchor
    FastChat LangServ(LangChain) PrivateGPT Tests
    Ray.io ollama React / NextJS QA/QC
    CyVerse Mistral MaterialUI
    Jetstream-2 OAuth

    Ansible Playbooks are used to deploy the software on OpenStack Cloud managed hardware

    K8s Helm Charts deploy the backend API

    NextJS and React are used for the https:// user interface UI

    OAuth & LDAP secure authentication are provided by CyVerse and University of Arizona.
